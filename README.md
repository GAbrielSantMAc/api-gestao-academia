# Sistema de Gestão de Academia

API REST desenvolvida utilizando Django e Django REST Framework.

## Funcionalidades

### Alunos

* Cadastrar aluno
* Listar alunos
* Editar aluno
* Excluir aluno

### Matrículas

* Cadastrar matrícula
* Listar matrícula
* Editar matrícula
* Excluir matrícula

## Tecnologias Utilizadas

* Python 3.11
* Django 5.2
* Django REST Framework
* SQLite

## Rotas da API

* /api/alunos/
* /api/matriculas/
